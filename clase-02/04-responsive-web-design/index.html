Mobile First

  • Fluid Design

    Utiliza porcentajes para definir el tamaño de los elementos, permitiendo que esto se adapten al contenedor, pero en pantallas grandes los mismos se estiran demasiado.
    También existen una variante llamada Liquid Design que establece el uso de maximos y minimos, pero esto genera espacios demasiados grades entre elementos.

  • Adaptive Design

    Consiste en diseñar una pagina para que cada tipo de dispositivo en el que se visualice el sitio, pero este caso los contenidos se repiten y son independentes uno de otro.
    Esta propuesta es ideal para clientes con bajo presupuesto, ya que generalmente se utilzian 2 versiones de sitio principal (desktop) y se utilizan herramientas para el redireccionamiento.

  • Resposive Design

    Diseño utilizando multimedia flexible, un sistema de grillas adaptable e impelemtenta el concepto one web for all para evitar la duplibicidad de contenido.
    Este tipo de diseño debe ser planificado desde la vista mobile y de ser posible utilizando los contenidos finales para lograr un mejor resultado a la hora de integrar los contenidoas.

Etiqueta Viewport

¿Cual es mi tamaño de viewport? MDN Meta tag
  • initial-scale: Establece el zoom inicial de la pagina normalemente 1
  • minimum-scale y maximum-scale: Establece nivel maximo y minimo de zoom (1 a 10)
  • user-scalable: Define si el usuario podra o no hacer zoom sobre el pagina (1, 0, no, yes)
